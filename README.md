# My Python Azure App 🚀

## 📌 Description

A simple Flask-based web application deployed on Microsoft Azure.

## ⚙️ Tech Stack

* Python
* Flask
* Gunicorn
* Microsoft Azure

## ▶️ How to Run Locally

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## 🌐 Endpoints

* `/` → Home route
* `/health` → Health check

## ☁️ Deployment

Deployed using Microsoft Azure App Service.
